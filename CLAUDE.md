# Digital Sorcery — Project Instructions

## Required Skills

Always use these skills for ALL development work on this project:

- **`/frontend-design`** — For all UI component creation, page design, and visual implementation
- **`/ui-ux-pro-max`** — For design system decisions, spacing, hierarchy, color, typography, and accessibility audits
- **`superpowers`** — For implementation planning, execution, debugging, code review, and git workflow

These skills are NOT optional. Invoke them before starting any design or development task.

## Design System

- **Golden ratio / Fibonacci**: ALL spacing, typography, and layout proportions derived from φ (1.618)
- **Spacing tokens**: `--fib-1` (8px) through `--fib-8` (233px)
- **Typography tokens**: `--text-fib-xs` (13px) through `--text-fib-2xl` (89px)
- **Background**: Deep obsidian `#050508`
- **Cards**: Prismatic glass with animated shimmer (`.refractive-card`, `.prismatic-glass`)
- **Warm accent**: `#E3CCB1` for labels, tags, verdicts
- **Crystalline silver**: `#c8d0e0` for borders, glows, shimmer
- **Hero accent**: `--color-ds-headline-accent` (white/silver)
- **No theme switcher** — dark only

## Brand Rules

- **BlackMagickOps** = Platform Engineering & Automation (NOT DevOps)
- **Digital Sorcery** = AI Intelligence Layer
- Never mix services between brands
- Never use "CAIO" or "Chief AI Officer"
- Never limit to specific verticals — ALL industries
- No consultant buzzword soup — ground everything in real problems

## Tech Stack

- Next.js 15 (App Router), React 19, TypeScript
- Tailwind CSS 4, Framer Motion
- React Three Fiber (WireframeOwl, ShaderLines)
- Icons: Lucide React (NOT emojis)
- Deployment: Netlify
